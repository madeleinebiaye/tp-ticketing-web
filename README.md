# TP Ticketing Web

Application web de gestion de ticketing – TP fil rouge du module Développement Web.

## Objectif
Développer progressivement une application web complète permettant de gérer :
- des clients
- des projets
- des tickets
- le suivi du temps
- la facturation

## État du projet
- Étape 1 : HTML / CSS (pages statiques)
- Navigation fonctionnelle
- Structure prête pour les prochaines étapes (JS, PHP, Laravel)

## Technologies utilisées
- HTML5
- CSS3
- JavaScript (à venir)
- PHP (à venir)
- Laravel (à venir)
